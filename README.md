# ReactTube
A YouTube clone that returns search results automatically. Built using React

# To run:
In the terminal, run this command:
```
npm start
```

The application will be accessible on http://localhost:8080

# Technologies used
Front-end:
- ReactJS
- HTML5
- CSS3

Packages:
- YTSearch
- Webpack
- Babel
- Lodash

API:
- YouTube API
